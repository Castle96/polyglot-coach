# Text To Speech Models

Polyglot Coach uses Piper TTS as the primary local speech synthesis engine.

## Supported Voices and Locales

- es_ES
- es_MX
- fr_FR
- fr_CA
- de_DE
- ja_JP

## Download

Download Piper voice models for the required locales and place the `.onnx` files in this directory.
Voice model files are ignored by git.
